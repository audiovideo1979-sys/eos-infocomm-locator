## Project Basics
- **Type:** React + Vite PWA with Firebase Realtime Database
- **Stack:** React 19, Vite 8, Tailwind CSS 4, Zustand, Firebase 12, vite-plugin-pwa
- **Purpose:** Real-time team locator app for the InfoComm trade show — tracks team member locations on hall maps, shows roster, schedule, and booth search
- **Deploy target:** Firebase Hosting (via `firebase.json`)
- **Dev server:** `npm run dev` → Vite on localhost:5173
- **Build:** `npm run build` → dist/
- **Push to GitHub:** `git add -A && git commit -m "..." && git push`
- **Working directory:** `C:\Projects Folder\eos-infocomm-locator\`

## Architecture

### Views (bottom nav)
- `map` → `MapView` — hall floor plan with team member location pins
- `roster` → `RosterPanel` — list of all team members + status
- `search` → `SearchPanel` — booth/exhibitor search
- `schedule` → `ScheduleView` — show schedule

### Key Files
- `src/App.jsx` — root, view router, toast notifications
- `src/store/useLocationStore.js` — Zustand store (all app state)
- `src/firebase/config.js` — Firebase app init
- `src/firebase/auth.js` — Firebase auth helpers
- `src/hooks/useFirebaseSync.js` — real-time Firebase RTDB sync
- `src/hooks/useGeolocation.js` — GPS location hook
- `src/hooks/useNotifications.js` — push notification listener
- `src/data/booths.js` — booth/exhibitor data
- `src/data/halls.js` — hall definitions (Central, North)
- `src/data/zones.js` — zone/area definitions
- `src/data/schedule.js` — show schedule data
- `src/data/pois.js` — points of interest
- `src/data/colors.js` — team color palette
- `public/` — hall map images (light + dark variants), PWA icons, manifest

### Components
- `JoinScreen` — name entry / team join on first load
- `Header` — top bar with title + controls
- `StatusBar` — shows active member count / status
- `Footer` — bottom nav (map/roster/search/schedule)
- `map/MapView` — floor plan + location pins
- `roster/RosterPanel` — team member list
- `search/SearchPanel` — exhibitor/booth search
- `schedule/ScheduleView` — schedule list
- `Toast` — notification popups
- `IOSInstallPrompt` — PWA install banner for iOS
